# E-commerce Sales Analytics using SQL, Python, and Tableau

## Overview

This project presents an end-to-end sales analytics workflow for an e-commerce store. It includes SQL data extraction from Google BigQuery, exploratory data analysis (EDA), statistical analysis in Python, and interactive dashboard development in Tableau.

---

## Objectives

- Extract and prepare data using SQL.
- Perform exploratory and statistical analysis.
- Analyze sales performance and customer behavior.
- Visualize key metrics in Tableau.
- Generate business insights.

---

## Dataset

The dataset was created by joining multiple tables from the Google BigQuery e-commerce dataset.

It contains information about:

- orders
- customers
- products
- traffic sources
- devices

A detailed description of the dataset is available in the project notebook.
